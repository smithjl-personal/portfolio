The Game of Life

This is a game John Conway came up with in 1970. The game has a two dimensional grid of live/dead cells, and each cell interacts with its eight neighbors. There are only four rules in the game. See the demo or the wiki page for more detail.

With only these four rules, incredible things can happen. This is a beautiful example of emergence; where complex or intricate systems can arise from simple ones. In other words, the whole is greater than the sum of its parts.

I created this with P5.js to help draw on the canvas, Bootstrap for styling some of the controls, and raw HTML/CSS/JavaScript for the rest. Check out the site, a live demo is available.

Chess Engine

I have heard many good things about Rust (the programming language). I decided I needed to see what the hype was for myself. I also happen to have a passion for chess. So the only logical thing to do was learn Rust and build a chess engine from scratch! The bot is playing as black in the images.

The engine uses bitboards, alpha-beta pruning, zobrist-hashes, lookup tables, iterative deepening, and many more features. It also interacts with the Lichess Bot API; so the engine can be played against on their site with their UI. The engine is also asynchronous, so it can play multiple games at once.

Sadly the bot is not running 24/7 because, well, servers aren't cheap. I'm happy to run the bot on my machine for anyone interested in facing it though! But the readme in the repository provides details on how to run the bot with other accounts if needed.
