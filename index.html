About Me

I'm a passionate lifelong learner who is currently working as a Senior Full-Stack Software Developer. I see something new or interesing, like a new language or framework, then I build a project in it for fun!

This website for example, was created because I realized I didn't really have anywhere to show off what I can do. It is written in raw html/css/javascript, and code that is pushed to the main branch of the project is automatically deployed. You can inspect the source code for it here if you are curious.
